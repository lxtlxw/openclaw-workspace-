# POE2 双控项目逻辑分析

## 项目概述
这是一个《流放之路2》（POE2）的游戏辅助工具，支持双人同屏控制、自动化操作和网络多机协同。

---

## 核心组件

### 1. 主程序文件
- **byspfxwshd.exe** (69.6 MB) - 主程序，负责游戏控制和自动化
- **PlugTool.ini** - 插件配置文件
- **HidHide-隐藏虚拟驱动设备.exe** (8.08 MB) - 隐藏虚拟设备防检测
- **游戏手柄-多联机映射延迟安装.exe** (6.28 MB) - 手柄映射延迟安装

### 2. 配置文件
- **egugyku5jv.json** (225 B) - 网络授权配置
- **w7x2xr9pmj.json** (44.6 KB) - 主配置文件（键位映射、自动任务等）

### 3. 安装包
- **5.0.zip** (74.6 MB) - 版本 5.0 安装包
- **5.1.zip** (74.6 MB) - 版本 5.1 安装包

### 4. 字库文件
- 字符识别模板（P1/P2 角色标识）
- 鼠标位置映射
- OCR 字典

### 5. 火绒安全组件 (HR/Huorong/Sysdiag)
- 火绒系统诊断工具集成
- 可能用于安全绕过或保护

## 核心逻辑分析

### 1. 双人同屏控制 (w7x2xr9pmj.json)

#### 键位映射 (Player1 / Player2)
```
Player1 (WASD移动):
- W/S/A/D → 左摇杆上下左右
- C → A键
- Space → B键
- Delete → Y键
- End → X键
- PageDown/PageUp → LB/RB
- LCtrl/Home → LT/RT
- N/M → LS/RS
- F → Back键
- Pause → Start键
- 数字小键盘 8546 → 方向键

Player2 (IKLJ移动):
- I/K/J/L → 右摇杆上下左右
- X → A键
- Space → B键
- 方向键 → 左摇杆（覆盖 WASD）
```

#### 双控切换
- **DualMuteButton**: LT+LS (静音切换)
- **DualMuteButtonP1/P2**: 独立静音按钮
- **SwapPadsHotkey**: 交换手柄映射

---

### 2. 自动化功能

#### TimedTasks (定时任务)
1. **水银药剂循环F5F6** - 循环使用药剂
2. **状态药剂F3** - 自动补充状态药
3. **buff技能1/2** - Buff 技能自动释放

#### ColorTasks (颜色触发任务)
1. **单人自动下鸟** - 检测特定颜色自动按空格
2. **自动酊剂5** - 检测冷却条颜色自动按 D5

#### AutoPotionTasks (自动药剂)
- **生命值P1/P2**: HP < 80% 自动吃药
- **魔力值P1/P2**: MP < 20% 自动补蓝
- **流放1生命/魔力/护盾**: 流放模式下的自动回血/蓝/盾

#### MacroTasks (宏任务)
- **按键宏1**: 按键位映射，支持循环和抖动延迟

---

### 3. 自动跟随 (AutoFollow)

#### P1 跟随配置
- **目标检测**: 右上角区域检测角色标识
- **跟随距离**: 70 像素
- **惯量**: 600ms
- **滚动**: 远距离自动按 B 键滚动
- **检测方式**: 颜色匹配 + 模板匹配 (P1: 28E3FB/2C65E6)

#### P2 跟随配置
- **目标检测**: 同上
- **检测颜色**: DFEC18/C82B18
- **其他参数同 P1**

---

### 4. 网络多机协同 (Lan)

#### 网络配置
- **主控 IP**: 10.0.0.128
- **从机列表**:
  - DESKTOP-P7F4FTO: 10.0.0.231
  - DESKTOP-9K3AVDK: 10.0.0.224

#### 局域网宏任务
1. **P1和P2交互键** - 双机同步按键
2. **传送到队友** - 同步传送操作

#### 切换功能
- **FollowToggleHotkey**: E (Ctrl+E 切换跟随)
- **SlaveSyncToggleHotkey**: Q (Ctrl+Q 切换从机同步)

### 5. OCR 字符识别

#### 字库文件
- **3818x2044.txt**: OCR 字典文件

- **鼠标.txt**: 鼠标位置映射

格式: ID$起始-结束$十六进制颜色数据
示例: 3$12-20-61$3F040880C00600600600400C0080180100200600C0180300600FFFFFF

## 安全与反制

1. **HidHide 驱动**: 隐藏虚拟输入设备
2. **火绒集成**: 可能利用火绒白名单机制
3. **网络授权**: 服务器验证，防止破解
4. **随机抖动**: DelayJitterMs 增加随机性

---

## 使用场景

1. **单机双控**: 一台电脑双人同屏玩
2. **多机协同**: 局域网多台电脑同步操作
3. **自动跟随**: P2 自动跟随 P1
4. **自动回血/蓝**: 低于阈值自动吃药
5. **Buff 管理**: 定时释放 Buff 技能

---

## 潜在问题

1. **网络依赖**: 需要连接授权服务器才能使用
2. **驱动安装**: HidHide 需要管理员权限安装驱动
3. **游戏更新**: OCR/颜色匹配需要根据游戏更新调整
4. **检测风险**: 虚拟手柄可能被高级反作弊检测

---

## 改进建议

1. **离线模式**: 去除网络授权依赖
2. **配置化**: 将 OCR/颜色配置外置，方便适配不同分辨率
3. **机器学习**: 使用模板匹配替代硬编码颜色
4. **云端同步**: 支持配置云同步
