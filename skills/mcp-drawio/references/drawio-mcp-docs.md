# Draw.io MCP Documentation

## What is Draw.io

Draw.io (now diagrams.net) is a popular open source diagramming tool used by millions for:
- System architecture diagrams
- Flowcharts
- ER diagrams
- Network diagrams
- UML diagrams
- And more

## What is Draw.io MCP

Draw.io MCP is the official MCP server that enables AI to:
- Generate diagrams from natural language descriptions
- Convert between diagram formats
- Create editable diagrams directly from AI

## Benefits

- **No manual drawing**: AI does the layout for you from description
- **Editable output**: You get native Draw.io XML that you can open and edit in Draw.io
- **Multiple formats**: Mermaid, CSV → Draw.io
- **Fast iteration**: AI can quickly adjust based on feedback
