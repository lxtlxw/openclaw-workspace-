# Context7 MCP Documentation

## What is Context7

Context7 brings real-time up-to-date library documentation to AI coding assistants. It solves the problem of AI models having outdated training data by providing:
- Always up-to-date documentation
- Semantic search across library documentation
- Working code examples directly from official docs

## Supported Libraries

Currently supports **9000+** popular libraries from:
- npm (JavaScript/TypeScript)
- PyPI (Python)
- And more

## How It Works

1. AI asks about a library
2. Context7 MCP searches its vector database
3. Returns the most relevant documentation for the library + version
4. AI uses the up-to-date documentation to answer correctly

## Benefits Over Static Knowledge

- ✅ No outdated API suggestions
- ✅ Works with newly released libraries
- ✅ Gets better as libraries update
- ✅ Semantic search finds exactly what you need
