# TrendRadar MCP Documentation

## What is TrendRadar

TrendRadar is an AI-powered public opinion monitoring and hot trend aggregation service. It collects hot topics from 11+ Chinese internet platforms and provides:
- Real-time hot topic aggregation
- AI-powered sentiment analysis
- Trend tracking and prediction
- Keyword monitoring
