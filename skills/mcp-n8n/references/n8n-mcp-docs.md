# n8n MCP Server Documentation

## What is n8n

n8n is a popular open source workflow automation tool that lets you connect different services and automate processes. It has:
- 500+ pre-built integrations
- Visual workflow editor
- Self-hostable
- Free and open source (72K+ GitHub stars)

## What is n8n MCP Server

n8n MCP Server enables AI to:
- Create workflows from natural language descriptions
- Manage existing workflows
- Trigger workflow execution
- Check execution status
- Manage credentials securely

## Why use AI + n8n

- AI understands your automation requirement in plain language
- AI automatically connects the right nodes in the right order
- You don't need to manually drag-and-drop in the UI
- Fast prototyping of automation ideas

## Getting n8n

- Self-hosted: https://github.com/n8n-io/n8n
- Cloud: https://n8n.io/cloud/
- Get API key from n8n Settings → API
